スマートホーム開発

小さく作って、確かに動く。

部屋の温湿度を計測し、リアルタイムで可視化。 さらにESP32で家電を遠隔操作できる、Raspberry Piベースのスマートホームシステム。

  • 温湿度センサー（SHT31）でリアルタイムデータを取得
  • InfluxDB＋Grafanaでグラフ化
  • ESP32＋赤外線モジュールでエアコン制御
  • HTTP通信でRaspberry Piと連携

🧠 使用技術: Raspberry Pi / ESP32 / InfluxDB / Grafana / Python / Arduino
