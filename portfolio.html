Smart Home Project

ESP32 × Raspberry Pi × Grafana

部屋の「今」を見える化し、心地よさを自動で最適化する。 温湿度センサー、データ可視化、電子暖炉/加湿器の自動制御を組み合わせた小規模スマートホームシステム。

デモを見る 技術を見る

System Overview

Raspberry Pi

データ収集と可視化を担当。Pythonスクリプトでセンサー値をInfluxDBへ記録。

ESP32

赤外線モジュールを通じて家電を制御。HTTP通信でラズパイと連携。

SHT31 Sensor

室内の温湿度を高精度に取得し、環境変化を即時にフィードバック。

Technical Details

🧠 Raspberry Pi

Pythonでセンサー値を取得し、InfluxDBへ自動記録。Grafanaで可視化ダッシュボードを構築。

📡 ESP32

HTTP経由でラズパイと通信。赤外線LEDでエアコンを制御し、快適温度に調整。

🌡️ SHT31

I2C通信でラズパイに接続し、温度・湿度をリアルタイム取得。

💾 InfluxDB + Grafana

ローカル環境での時系列データ蓄積とダッシュボード化を実現。

Challenges & Learnings

  • ESP32とRaspberry Pi間のHTTP通信で発生する遅延を軽減。
  • センサー値のノイズを平滑化するために平均化アルゴリズムを実装。
  • GrafanaのUIを「ホテルダッシュボード」を意識したデザインに調整。
  • 家庭用電源環境でも安定稼働するよう電力供給を最適化。
