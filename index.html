スマートホーム × エッジ開発が得意です。

Raspberry Pi（InfluxDB/Grafana）と ESP32（赤外線制御）を用いて、
「計測 → 蓄積 → 可視化 → 家電制御」までを一体化したシステムを開発しています。

ESP32 Raspberry Pi InfluxDB Grafana MQTT/HTTP IR Remote Python C++/Arduino
ポートフォリオを見る

やっていること

  • 温湿度などのセンサーデータを InfluxDB に時系列保存
  • Grafanaでのダッシュボード設計（アラート/テキストパネル/UI）
  • ESP32で赤外線（IR）制御：エアコン/加湿器/TV
  • Python スクリプトの常駐化（systemd/cron）

目指していること

滞在体験を高める“見える・触れる”スマートホーム。
現場に設置できる堅牢さと、見せても映えるUIの両立を重視します。
